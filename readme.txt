# University Theme

## Theme Overview

University Theme is a custom WordPress theme developed specifically for the assessment project. The theme follows WordPress Coding Standards (WPCS), PHP 8 compatibility, and WordPress best practices.

---

### Theme Development

* Custom WordPress Theme
* Dynamic Header
* Dynamic Footer
* Custom Logo Support
* Primary Navigation Menu
* Footer Navigation Menu
* Responsive Layout
* Translation Ready
* WordPress Coding Standards (WPCS)

## Custom Post Type

**Courses**

Implemented using WordPress Custom Post Type API.

Supports:

* Title
* Editor
* Featured Image
* Excerpt
* Archive
* REST API

---

## Taxonomy

Course Categories
